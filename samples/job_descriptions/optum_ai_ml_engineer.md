# AI/ML Engineer - Optum

## Role Summary

AI/ML Engineer building generative AI and machine learning capabilities for
cloud-native digital products. The role emphasizes Python development,
LLM-powered applications, agentic frameworks, responsible AI practices, and
collaboration with engineering and product stakeholders in digital/eCommerce
or adjacent customer-facing environments.

## Responsibilities

- Build and improve machine learning and generative AI solutions using Python.
- Develop LLM-based workflows with OpenAI API, LangChain, LangGraph, AutoGen,
  and related agentic AI frameworks.
- Prototype and integrate cloud-native AI capabilities using services such as
  Amazon Bedrock, Vertex AI, and Azure where appropriate.
- Work with NLP, embeddings, transformers, and Hugging Face tooling to support
  retrieval, summarization, classification, or conversational use cases.
- Apply responsible AI and ethical AI practices, including transparency,
  evaluation, safety checks, and clear communication of model limitations.
- Use modern AI-assisted development tools such as GitHub, VS Code, GitHub
  Copilot, Codex, and Claude Code to accelerate engineering workflows.
- Partner with product, design, engineering, and business stakeholders to turn
  AI opportunities into reliable digital experiences.

## Required Skills

- Python.
- GitHub and collaborative software development workflows.
- OpenAI API.
- LangChain, LangGraph, or AutoGen.
- Hugging Face Transformers.
- Amazon Bedrock or Vertex AI.
- VS Code with GitHub Copilot, Codex, or Claude Code.
- Machine learning, NLP, LLMs, and generative AI.
- Cloud-native application development.
- Responsible AI, ethical AI, model evaluation, and safe deployment practices.

## Preferred Skills

- Communication skills and ability to explain technical concepts to
  cross-functional stakeholders.
- Digital/eCommerce or customer-facing product experience.
- Azure.
- Experience with agentic frameworks and AI-assisted engineering workflows.
