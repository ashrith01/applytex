## Role Summary

Machine Learning Engineer supporting data management and analytics solutions for
defense and intelligence customers. The role emphasizes production ML systems,
model deployment, data pipelines, API integration, and collaboration across data
science, software engineering, and product stakeholders.

## Responsibilities

- Design, build, and implement ML models and algorithms for business problems.
- Build scalable ML pipelines for ingestion, preprocessing, feature engineering,
  model training, evaluation, deployment, and monitoring.
- Convert ML models into deployable APIs and integrate them with existing
  applications and infrastructure.
- Evaluate model performance, iterate on accuracy and efficiency, and maintain
  reliability in production.
- Work with large datasets and potentially big-data processing technologies.
- Document models, experiments, deployment processes, and team best practices.

## Required Skills

- 3+ years of ML, computer science, data science, or related industry experience.
- Python.
- TensorFlow, Keras, PyTorch, or scikit-learn.
- Regression, classification, clustering, dimensionality reduction, and deep
  learning.
- Data preprocessing, feature engineering, and visualization.
- SQL, NoSQL, Spark, or Hadoop.
- AWS, Azure, or GCP ML services.
- Git and CI/CD.
- Communication and collaboration skills.

## Preferred Skills

- NLP, computer vision, or recommender systems.
- MLOps and model workflow automation.
- Docker and Kubernetes.
- RESTful API deployment.
- Distributed computing.
- Statistical modeling and inference.
